HTTP Status Codes giúp:

client biết request thành công hay thất bại
phân loại lỗi rõ ràng
xử lý logic phù hợp phía frontend/mobile

Khi trả null:
client không biết lỗi gì xảy ra
có thể vẫn nhận 200 OK
frontend khó xử lý lỗi

Dependency này giúp:
convert Java Object → XML
hỗ trợ Content Negotiation
